# Planning Poker

A real-time planning poker application built with Cloudflare Workers, Durable Objects, and React.

## Overview

Planning Poker (also known as Scrum Poker) is a collaborative estimation technique used by agile teams. This application provides a seamless, real-time experience for distributed teams to estimate work together.
